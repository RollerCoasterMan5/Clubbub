Installation & Important notes - READ ME

NOTE: This documentation assumes you have basic understanding in HTML & CSS.

- First download & include Bootstrap & Font Awesome Icons (for Icons).

- Now include our base css file ashobiz.base.css. It has basic styles like colors, buttons, labels, etc.,.

- Ashobiz has separate stylesheet for each component. To include a component just include ashobiz-X.css (where X denotes component number). You can include as many components as you need.

<!-- CSS FILE EXAMPLE -->

<!-- Ashobiz base -->
<link href="ashobiz.base.css" rel="stylesheet">

<!-- Component 1 -->
<link href="ashobiz-1.css" rel="stylesheet">
<!-- Component 5 -->
<link href="ashobiz-5.css" rel="stylesheet">

- To avoid conflicts, we use prefix a- in all our classes. Do remember.

- Ashobiz supports the following colors - white, grey, black, red, green, lblue (light blue), blue, orange, yellow, violet, rose and brown. Remember these color names since we use them frequently.

- LESS file is provided for all components. ashobiz.base.less is the base file which has basic styles like colors, background, buttons, etc.,

Text Color

To add color to any text, use the class a-COLORNAME. For example, for Red text color use a-red.

Background Color

To add background color to any item, use the class a-bg-COLORNAME. For example, for Green background color use a-bg-green.

Labels

Ashobiz supports labels in different colors and shapes.

Labels

To add labels you need to use two classes a-label & a-label-COLORNAME. For example, to add rose label use a-label a-label-rose.

Label Label Label Label Label Label Label Label Label Label Label Label

Icon Labels

For icon labels, you need to add an extra class a-label-icon along with above mentioned classes. Font awesome icons plugin is must.

Buttons

Ashobiz supports buttons in different sizes, styles and colors. You can use the following button classes in anchor links <a>, form input tags <input> and form button tags <button>.

Normal Buttons

To create a button you need to use two classes a-btn and a-btn-COLORNAME.

Button Button Button Button Button Button Button Button Button Button Button Button

Small Buttons

For small buttons you need to use the class a-btn-sm.

Button Button Button Button Button Button Button Button Button Button Button Button

Large Buttons

For large buttons use the class a-btn-lg.

Button Button Button Button Button Button Button Button Button Button Button Button

Icon Buttons

Like Icon Labels for Icon buttons you need to use the class a-btn-icon. Font awesome icons plugin is must.

Social Media

Currently Ashobiz supports facebook, twitter, google plus, linkedin and pinterest icons.

Square social media icons

For social media icons you need to use two classes a-social and a-facebook or a-twitter or a-google-plus or a-linkedin or a-pinterest. Font awesome icons plugin is must.

Transparent social media icons

For transparent social media icons use the class a-social-trans.

Border

Ashobiz let you easily add colored borders to any items with some simple classes. It also have classes for border width and styles.

Border Colors

Use the classes a-br and a-br-COLORNAME to add border. By default it adds border with 1px width.
